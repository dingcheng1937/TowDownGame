## 二、核心代码解析

### 2.1 怪物属性

```gdscript
extends CharacterBody2D
class_name BaseMonster

@export var SPEED = 50.0
@export var HP = 5
@export var hurt = 1
var is_die = false
var target_player = Utils.player
```

### 2.2 AI移动

```gdscript
func _physics_process(delta):
    if target_player != null:
        var direction = global_position.direction_to(target_player.global_position)
        velocity = direction * SPEED
        move_and_slide()
```

### 2.3 伤害处理

```gdscript
func onHit(hit_num):
    HP -= hit_num
    if HP <= 0:
        onDie()

func onDie():
    is_die = true
    PlayerData.player_exp += 1
    queue_free()
```

## 三、代码练习题

### 练习1：巡逻行为

```gdscript
var patrol_points = []
var current_index = 0

func patrol():
    var target = patrol_points[current_index]
    velocity = global_position.direction_to(target) * SPEED * 0.5
    if global_position.distance_to(target) < 30:
        current_index = (current_index + 1) % patrol_points.size()
```

### 练习2：掉落系统

```gdscript
@export var drop_table = {"gold": 0.8, "hp_pack": 0.3}

func onDie():
    for item in drop_table:
        if randf() < drop_table[item]:
            spawn_drop(item)
```
