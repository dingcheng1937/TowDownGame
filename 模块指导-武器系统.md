## 二、核心代码解析

### 2.1 武器属性

```gdscript
extends Node2D
class_name BaseGun

@export var damage = 0.0
@export var fire_rate = 5.0
@export var bullets_max_count = 10
@export var recoil = 0

var bullets_count = 0
var is_use = false
```

### 2.2 射击流程

```gdscript
func _process(delta):
    if Input.is_action_pressed("shoot") and can_shoot:
        can_shoot = false
        timer.start()
        if bullets_count > 0:
            _shoot()
        else:
            reload_ammo()
```

### 2.3 换弹流程

```gdscript
func reload_ammo():
    if !is_reloading && bullets_count < bullets_max_count:
        is_reloading = true
        change_timer.start(change_speed)
```

## 三、代码练习题

### 练习1：三连发功能

```gdscript
@export var burst_count = 3

func fire():
    for i in burst_count:
        bullets_count -= 1
        var bullet = bullet_scene.instantiate()
        bullet.fire()
        if i < burst_count - 1:
            await get_tree().create_timer(0.1).timeout
```

### 练习2：武器过热

```gdscript
var heat = 0
var max_heat = 100

func _shoot():
    heat += 10
    if heat >= max_heat:
        can_shoot = false
    # 冷却逻辑
```
