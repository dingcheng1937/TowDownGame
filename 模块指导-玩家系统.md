## 二、核心代码解析

### 2.1 玩家类定义

```gdscript
extends CharacterBody2D
class_name Player

@onready var anim = $body/AnimatedSprite2D
@onready var gun_root = $body/GunRoot
var gun = null
var SPEED = 100.0
var is_dead = false
var is_dash = false
```

### 2.2 移动控制

```gdscript
func _physics_process(delta):
    var direction = Input.get_vector("left", "right", "up", "down")
    velocity = direction * SPEED
    if is_dash:
        velocity = direction * 600
    move_and_slide()
```

### 2.3 冲刺机制

```gdscript
func _input(event):
    if Input.is_action_just_pressed("dash") && !is_dash:
        is_dash = true
        await get_tree().create_timer(0.06).timeout
        is_dash = false
```

### 2.4 伤害处理

```gdscript
func onHit(hurt):
    PlayerData.player_hp -= hurt
    Utils.showHitLabel(hurt, self)
```

### 练习1：添加二段跳

```gdscript
var jump_count = 0
const MAX_JUMPS = 2

func _input(event):
    if Input.is_action_just_pressed("jump") && jump_count < MAX_JUMPS:
        velocity.y = -300
        jump_count += 1
    if is_on_floor():
        jump_count = 0
```
